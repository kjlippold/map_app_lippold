Assignment Instructions
  1. Build a new Tethys app called "mapapp" on your Ubuntu server, using the scaffold command.
  2. Remove all of the default content of the app while keeping the side bar, bottom, top, and main app areas - just remove the excess HTML.
  3. Create 4 links on the sidebar:
    1. Home
    2. Map View
    3. Data Services
    4. About
  4. Each of these links should open a new page in the main view.
    1. Home should open a "home page" with the name of the class, name of your app, date, assignment number and main assignment requirements listed.
    2. Map View should include an ArcMap Javascript API map with two data sets from your ArcMap Geoserver2 server, toggle buttons to turn off and on a raster layer and vector layer, and popup bubbles with information about the features in your map.
    3. Data Services will be a page that talks about the data services you have used in your map, how you created the data, where you are hosting the data, what data is included, and any additional data about your data. Also include a link to the geoserver2 data view page for your data.
    4. About - the about page will include a picture of you, a link to send you an email, and a little bit about who you are.
  5. Submit a document with screenshots showing all required elements running on your computer.
  6. Be prepared to do a quick 2 minute demo of your app in class on Tuesday.
